This directory contains the code for the QBF solver pgbddq.

PGBDDQ:
	pgbddq.py    The main solution code
	bdd.py	      A proof-generating BDD package
	proof.py      Generates proofs in QPROOF and QRAT proofs
	resolver.py   Support for generating resolution proofs
	util.py       Miscellaneous support routines

   Run ./pgbddq.py -h to see the commandline options
